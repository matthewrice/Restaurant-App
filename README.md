# Restaurant App
### https://matthewrice.github.io/Restaurant-App/
### Unit: React Review & Project Management
### Week: 7
### Tools: HTML, Bootstrap, SCSS, JavaScript, jQuery, Backbone, React

This app allows a user to view Majestic Thai's menu and place an order.

The objective of this assignment was to help me...
  - understand how to design and execute a front end application when given ambiguous requirements by a client.
  - understand how all of the pieces of Backbone + React fit together to create a dynamic frontend application.
